# mahjong_conscience

a basic riichi mahjong engine implemented in rust as an educational exercise.

## why "conscience"?

because [mahjong soul](https://mahjongsoul.yo-star.com/) is a popular riichi mahjong game on pc and mobile and [thesaurus.com](https://www.thesaurus.com/browse/soul) lists "conscience" as the strongest synonym for "soul".

## so this engine aims to have parity with mahjong soul?

i hope so, with the standard 4-player ruleset at least.

## since you're developing a mahjong engine, surely you must know the rules well, right?
